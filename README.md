# AI Snake Studio 🐍🤖

Welcome to **AI Snake Studio**, a production-quality educational reference implementation of a browser-based, AI-powered game. This project is designed to teach experienced C# developers how to build modern, local-first AI applications using the latest web technologies.

## 🚀 Project Overview

AI Snake Studio is a fully client-side application that runs a classic Snake game whose visuals and rules are generated by a local Large Language Model (LLM).

- **No Backend:** Everything runs entirely in your browser.
- **Local AI:** Inference happens on your device using WebGPU or WASM.
- **Data-Driven:** The AI generates structured JSON that the game engine interprets.

## 🛠️ Technology Stack

- **Frontend:** .NET 10 Blazor WebAssembly Standalone
- **Languages:** C#, JavaScript, HTML, CSS
- **AI Engine:** Transformers.js (utilizing ONNX Runtime Web)
- **AI Model:** SmolLM2-360M-Instruct (Local Inference)
- **Acceleration:** WebGPU (with WASM CPU fallback)
- **Rendering:** HTML5 Canvas
- **Testing:** xUnit
- **Deployment:** GitHub Pages via GitHub Actions

## 💡 Motivation & Learning Objectives

The goal of this project is to provide a "living textbook" for:
1. **WebAssembly (WASM):** Understanding how .NET executes in the browser sandbox.
2. **WebGPU:** Modern GPU compute for high-performance AI.
3. **Local AI Inference:** Running LLMs without expensive cloud APIs (Privacy & Cost).
4. **JS Interop:** Building bridges between C# and the JavaScript ecosystem.
5. **Clean Architecture:** Applying SOLID principles to game design.

## 📂 Project Structure

- `GameEngine/`: The heart of the app. Pure C# logic for Snake, Rules, and State.
- `wwwroot/js/`: The JavaScript bridge for AI (`ai.js`) and Rendering (`gameRenderer.js`).
- `Pages/`: Blazor components for the UI.
- `docs/`: Extensive educational documentation with Mermaid diagrams.
- `tests/`: xUnit test suite for the engine logic.

## 🎮 How to Play

1. Use **Arrow Keys** to control the snake.
2. Type a creative prompt in the **AI Studio Prompt** (e.g., *"A neon cyberpunk city with no walls and high speed"*).
3. Click **Apply AI Vision**.
4. Watch as the local AI reconfigures the game's colors, fonts, and physics in real-time.
