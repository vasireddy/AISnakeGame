# Reference Walkthrough

Welcome, C# Developer! This guide will help you navigate the AI Snake Studio codebase.

## 1. Dependency Injection
Look at `Program.cs`. We register `IGameConfigurationGenerator` here. In `Home.razor`, we use `@inject IGameConfigurationGenerator AIGenerator` to access the AI logic. This follows standard .NET best practices.

## 2. The Bridge
The `TransformersJsConfigGenerator` is the bridge. It calls `ai.js` using `IJSRuntime`.

## 3. Powerups Logic
Check `GameState.cs` and `RuleInterpreter.cs`. Notice how the `Update` loop handles spawning powerups based on the `Rules.Powerups` toggle generated by the AI.

## 4. Canvas Drawing
The `gameRenderer.js` handles the actual drawing. We pass the C# `GameState` and `Theme` objects as anonymous JS objects.
